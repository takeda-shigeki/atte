# atte
企業の勤怠管理システムである。<br>
![home_page](https://github.com/takeda-shigeki/atte/assets/132808377/4c46b58e-c305-42d5-a469-318824f16f39)

## 機能一覧
会員登録<br>
ログイン<br>
ログアウト<br>
勤務開始登録<br>
勤務終了登録<br>
休憩開始登録<br>
休憩終了登録<br>
日付別勤怠実績表示<br>
ユーザー別勤怠実績表示<br>

## 使用技術（実行環境）
Laravel 8.83.27<br>
PHP 8.2.11<br>
